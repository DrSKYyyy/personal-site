# 从 Obsidian 复制到博客的注意事项

## 正确格式模板

```yaml
---
title: 文章标题          # 必填
date: "2026-05-01"      # 必填，日期必须加引号
tags:                    # 可选
  - 标签1
  - 标签2
visibility: 公开         # 可选，默认"公开"。可选值：公开 / 私密 / 草稿
---
```

## 避坑清单

### 1. `date` 必须加引号

```yaml
date: 2026-05-01     # ❌ YAML 会自动解析成日期类型，报错
date: "2026-05-01"   # ✅ 加引号强制为字符串
```

### 2. 冒号用半角

```yaml
title：文章标题       # ❌ 全角冒号，YAML 不识别
title: 文章标题       # ✅ 半角冒号
```

### 3. `title` 字段不能少

内容集合要求 `title` 为必填，从 Obsidian 复制过来时要加上。

### 4. 分隔符是 `---`

```markdown
---                    # 开头：三个减号
title: xxx
date: "2026-05-01"
---                    # 结尾：三个减号

正文从这里开始
```

不要写成 `***` 或 `----------------`。

### 5. 注意文件名不要重复

文件名就是文章的 id，如果有两个文件同名（含中文转义后冲突），Astro 会警告。

---

**简单来说，每次从 Obsidian 复制一篇新文章到博客文件夹时，确保 frontmatter 长这样就行：**

```yaml
---
title: 标题
date: "2026-05-01"
tags:
  - 标签
---
```

其他字段可以省略。
